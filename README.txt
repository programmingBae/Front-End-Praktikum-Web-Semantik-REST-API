disediakan di Folder "DATABASE" kedalam database login yang telah 
dibuat di phpmyadmin
3. Setelah import databases login ke phpmyadmin maka 
bisa masuk ke http://localhost/login/
4. untuk masuk bisa dafar menjadi member menggunakan fitur create account.
5. lalu bisa masuk sebagai member.
6. ketika ingin login sebagai admin maka bisa login menggunakan
email : admin@gmail.com
password : 12345
7. setelah login menggunakan akun admin maka dapat login sebagai admin dan menggunkaan fitur yang dapat diaccess sebagai admin
